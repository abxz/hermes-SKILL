# 合同审查助手

## 一、Skill 用途
针对对上承包合同、对下劳务分包、对下材料采购三类合同及配套文件，开展全维度自动化审查。核心覆盖合同主体、关键数据、日期逻辑、资质合规、用章规范、报价单位关联关系六大维度，自动核验文件一致性、准确性、合规性，排查报价单位关联风险，内置全自动迭代机制持续优化审查能力，减少人工遗漏，适配三类合同标准化审查场景。

## 二、前置条件
1. 已启动 Hermes Agent，程序正常运行（本地读取文件，无需联网）
2. 用户已上传完整合同及配套文件，文件可正常读取、无损坏
3. 支持用户指定合同类型，未指定时自动识别并匹配对应审查标准
4. 含报价单文件时，需确保三家报价单位信息完整可提取

## 三、操作步骤（自动执行流程）
1. 接收用户上传文件，自动提取核心信息：甲乙双方名称、含税金额、税额、不含税金额、工期、文件日期、资质信息、用章审批单内容、报价单位信息。
2. 按文件功能自动分类（报价单、采购报告、评审表、合同主体、用章申请等），不限制文件数量、名称、编号。
3. 以合同主体为基准，对所有配套文件开展交叉核查。
4. 主体信息核查：所有文件甲乙双方名称与合同主体完全一致，无错别字、无简称/全称混用。
5. 金额核查：所有文件金额三要素（含税金额、税额、不含税金额）与合同主体一致，计算逻辑正确（不含税金额+税额=含税金额）。
6. 工期核查：所有文件工期、施工起止时间与合同主体完全一致。
7. 日期逻辑核查（仅对下劳务分包合同执行）：
   - 3份报价单日期 比非招标采购报告评审日期提前2天；
   - 非招标采购报告日期 比自身评审日期晚2天；
   - 合同线下评审表评审日期 = 非招标采购报告评审日期。
8. 未明确要求核查的空白日期，不做审查。
9. 资质与签名核查（按合同类型自动适配）：
   - 对下劳务分包：报价单需提供施工劳务资质+安全生产许可证（在有效期），签名可为法人代表本人签名或法人印章；
   - 对下材料采购：仅核查供应商营业执照（有效、经营范围匹配采购需求）；
   - 对上承包：核查承包方对应工程等级施工资质+安全生产许可证（在有效期），签名可为法人代表本人签名或法人印章。
10. 报价单位关联关系核查：
    - 提取三家报价单位的法人姓名、注册地址、联系邮箱、联系手机号；
    - 交叉比对：核查是否存在法人相同、注册地址一致、邮箱重复、手机号重复等关联情形；
    - 异常判定：出现任意一项重复/关联，标记为==报价单位关联风险==。
11. 用章审批单专项审查（三类合同通用）：
    - 印章勾选：必须勾选合同章、法定代表人印章，不勾选公司公章、法定代表人手签；
    - 用章事由：明确填写「XX合同签订用章」，开头空两格；
    - 审批栏：项目主管部门、市场开发部、分管领导、总经理、法定代表人审批/签字栏目无缺失；
    - 备注栏：仅核查栏目存在，不核查内容。
12. 审查报告生成：自动汇总通过项/问题项，问题项标注文件功能+具体错误，==报价单位关联风险==单独高亮标注，便于快速整改。

## 四、异常处理
1. 文件缺失：缺失核心功能类文件，自动提示用户补充，暂停审查直至文件齐全。
2. 信息无法识别：核心信息模糊、无法提取，标记为待确认，提示补充清晰文件或手动录入。
3. 审查异常：发现信息冲突、逻辑错误全程记录不中断，完成全部审查后统一输出整改清单。
4. 类型识别错误：支持用户手动修正合同类型，自动重新执行对应审查流程。
5. 报价单位信息缺失：无法提取法人/地址/邮箱/手机号，标记为信息不全待补充，不判定关联关系。
6. 报价单位关联异常：检测到关联风险，高亮提示并标注具体重复项。
7. 迭代异常：自动迭代失败时，保留当前稳定版本，不影响正常审查。

## 五、注意事项
1. 仅核查指定日期，空白日期不审查。
2. 资质审查仅针对报价单，合同主体不重复核查。
3. 用章审批单仅核查填写规范、勾选情况与栏目完整性，不核查审批意见与签字真实性。
4. 金额三要素必须同步核对，确保计算逻辑正确。
5. 自动识别文件功能，不受文件名称、编号影响。
6. 报价单位关联核查仅针对三家报价单位，无报价单文件不执行本项。
7. 关联判定以完全一致为标准，相似地址/邮箱做疑似风险标注。
8. 自动迭代全程后台运行，不干扰正常审查流程。

## 六、自动迭代机制（全自动执行）
1. 审查规则自动迭代：
   - 自动记录用户手动修正、自定义标准、例外场景；
   - 实时更新日期规则、资质要求、用章规范、关联判定阈值；
   - 三类合同规则自动差异化适配，无需手动修改脚本。
2. 问题库自动迭代：
   - 累计高频错误（日期冲突、资质缺失、关联风险等）；
   - 生成企业专属问题预警库，审查前提前拦截风险；
   - 按部门/合同类型分类统计，形成合规画像。
3. 识别能力自动迭代：
   - 持续优化文件分类、文字提取、信息识别准确率；
   - 提升扫描件/图片/模糊文件的读取成功率；
   - 优化报价单位信息提取，覆盖更多字段格式。
4. 报告模板自动迭代：
   - 根据用户整改习惯，自动优化报告结构与展示方式；
   - 关联风险、高频问题自动高亮置顶；
   - 新增整改建议自动匹配，无需人工补充。
5. 关联核查规则迭代：
   - 从完全匹配升级为完全匹配+疑似关联双重判定；
   - 支持地址模糊比对、邮箱前缀比对、手机号段比对；
   - 自动学习企业黑名单，快速识别高风险单位。
6. 版本自动归档与回滚：
   - 每次迭代自动保存版本，保留操作日志；
   - 迭代异常时自动回滚至上一稳定版本；
   - 支持查看迭代历史，追溯规则变更。

## 七、Skill 优化方向
1. 支持企业内部供应商库对接，一键校验报价单位资质。
2. 新增合同风险等级自动评定（低/中/高）。
3. 支持批量审查，一次性处理多份合同文件。
4. 生成月度/季度合规审查统计报表。
